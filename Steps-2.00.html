Dataset Information and Methodology 

1) The Dataset consists of 33 attributes and 569 rows with NO missing values according to the donor
2) All feature values are recorded with four significant digits. Class distribution: 357 benign, 212 malignant.
3) Ten real-valued features are computed for each cell nucleus:
    a) radius (mean of distances from the center to points on the perimeter)
    b) texture (standard deviation of gray-scale values)
    c) perimeter 
    d) area    
    e) smoothness (local variation in radius lengths) 
    f) compactness (perimeter^2 / area - 1.0) 
    g) concavity (severity of concave portions of the contour) 
    h) concave points (number of concave portions of the contour) 
    i) symmetry 
    j) fractal dimension ("coastline approximation" - 1)
The above Zip file consists of three folders having, Input, Output, and Scripts. The Input folder consists the data which has been utilized for the modelling, the Output folder contains the images which are resulted from the models and other techniques while processing and lastly, the scripts are the files in two formats (.py, .ipynb) for compiling in both Spyder and Jupyter Notebook.
