Step4: Data Preprocessing


Preprocessing is a data mining technique that involves transforming raw data into an understandable format. Data cleaning and Data transformation are performed in this step.

  •  Firstly, since the features such as 'id' have nothing to do with classification, diagnosis being a 'class' label and 'unnamed: 32' having NaN values, now by performing drop function, the redundant columns are removed from the current data frame.
  • Besides, all the values are converted into float (numerical) data types.


Thereby, all the data is transformed into numerical data type, it helps us for better analysis.
